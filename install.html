Setup und Tools

Webserver, PHP und IDE

PHP Laufzeitumgebung

$ sudo apt-get install nginx php5-fpm php5-cli php5-xdebug

Xdebug und PHP Konfiguration

; /etc/php5/fpm/conf.d/error_log.ini
error_reporting = E_ALL         ; Fehler-Level auf alle Fehler setzen
display_errors = On             ; Fehler direkt ausgeben (stdout)
display_startup_errors = On     ; Fehler beim Starten des PHP Prozesses ausgeben
html_errors = On                ; Schoen formatierte Fehlerausgabe
log_errors = On                 ; Fehler in eine Log-Datei schreiben
log_level = debug               ; Log-Level festlegen
error_log = /tmp/php_errors.log ; Ort der Log-Datei festlegen

; /etc/php5/fpm/conf.d/xdebug.ini
xdebug.remote_enable = true     ; Interface fuer entfernten Zugriff auf Xdebug einschalten

danach muss PHP neugestartet werden

$ sudo service php5-fpm restart

Versionsverwaltung mit Git

$ sudo apt-get install git


// Ein neues Repository anlegen
$ mkdir projektverzeichnis && cd projektverzeichnis
$ git init

// Dateien dem Staging-Bereich hinzufuegen
$ git add test.txt

// Vorbereitete Aenderungen (im Staging-Bereich) commiten
$ git commit -m "Das ist unser erster commit"

// Ein entferntes Repository hinzufuegen
$ git remote add origin …

// Aenderungen an ein Repositoy uebertragen
$ git push -u origin master

// danach reicht
$ git push

// Ein entferntes Repository clonen
$ git clone …

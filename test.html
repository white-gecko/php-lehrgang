Codeing Conventions und Testen

Code Schniffer installieren

// composer installieren
$ curl -sS https://getcomposer.org/installer | php

// php code sniffer isntallieren
$ php composer.phar global require "squizlabs/php_codesniffer=*"

// composer bin zum $PATH hinzufügen
PATH="$PATH:$HOME/.composer/vendor/bin/"
// in ~/.profile eintragen und neue terminal sitzung starten

// Coding Standards auschecken
$ git clone -b cs https://github.com/white-gecko/php-lehrgang.git

// Code Sniffer ausführen
$ phpcs --standard=CodeSniffer/Standards/OntoWiki/ --extensions=php -p application/controllers/

PHP Unit installieren

$ php composer.phar global require "phpunit/phpunit=*"

// Tests auschecken
$ git clone -b test https://github.com/white-gecko/php-lehrgang.git
